💡 UTILISATION RAPIDE
═════════════════════

Chargement minimal (débutant):
  $ source ~/ubuntu-configs/mon_shell/exemple_simple.sh
  $ check                    # Diagnostic
  $ aide-simple             # Voir commandes

Chargement complet (recommandé):
  $ source ~/ubuntu-configs/mon_shell/chargeur_modules.sh
  $ aide                    # Interface aide
  $ modules                 # Liste modules

Intégration permanente bash:
  Ajouter à ~/.bashrc:
  [ -f ~/ubuntu-configs/mon_shell/chargeur_modules.sh ] && \
      source ~/ubuntu-configs/mon_shell/chargeur_modules.sh

Intégration permanente zsh:
  Ajouter à ~/.zshrc:
  [ -f ~/ubuntu-configs/mon_shell/chargeur_modules.sh ] && \
      source ~/ubuntu-configs/mon_shell/chargeur_modules.sh
